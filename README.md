# Bristol Gloucester PREM Preview

Static GitHub Pages project for the Bristol Bears vs Gloucester Rugby Gallagher PREM match on April 17, 2026 at Ashton Gate.

Expected live URL after GitHub Pages is enabled:

https://zeshanalikhan.github.io/bristol-gloucester-prem-preview/

## Project notes

- Separate project folder and site implementation.
- Includes Google site verification meta tag.
- Includes a nofollow/sponsored bonus smartlink CTA above the fold.
- Uses internal navigation pages for preview, standings, head-to-head, live notes, and FAQ.
- Uses a local JSON file for the live notes board.
- Uses an edited, compressed Wikimedia Commons Ashton Gate image with visible attribution.
